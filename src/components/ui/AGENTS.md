# UI - shadcn/ui Primitives

Generated shadcn/ui primitives (e.g. `select.tsx`). Treat as vendored.

## Conventions

- Add new primitives with the shadcn CLI, do not hand-write them.
- App components import from here; do not put app-specific logic in these files.

## What Not To Do

- Do not hand-edit generated primitives beyond what shadcn supports - customizations belong in the consuming component.
